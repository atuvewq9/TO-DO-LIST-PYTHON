# TO-DO-LIST-PYTHON
A simple command-line To-Do List application built using Python.   This project allows users to manage daily tasks efficiently through a menu-driven interface.
# Features

- Add new tasks
- View all tasks
- Delete completed tasks
- Simple and beginner-friendly interface
- Runs continuously until the user exits

---

## Technologies Used

- Python
- Lists
- Loops
- Conditional Statements
- Functions

# Concepts Learned

- Python Lists
- Loops (`while`, `for`)
- Conditional Statements (`if-elif-else`)
- User Input Handling
- List Operations (`append()`, `pop()`)
- Menu-Driven Programming
